# egg初始化项目脚手架


## 自动生成接口文档
#### 1、全局安装apidoc
~~~
npm i apidoc -g
~~~
#### 2、执行指令生成接口文档
~~~
npm run apidoc
~~~
#### 3、启动项目后，直接访问public中的apidoc项目
#### 如：http://127.0.0.1:7001/public/apidoc/index.html


## 开发
```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

## 线上发布

```bash
$ npm start
$ npm stop
```
